# Calculadora en C# (Unity)

Proyecto práctico desarrollado en **C#** utilizando **Unity**, enfocado en la práctica de programación orientada a objetos y gestión de eventos en interfaz de usuario.

## Funcionalidades

- Operaciones básicas: suma, resta, multiplicación y división
- Función de limpiar / reiniciar
- Interacción mediante botones y eventos
- Estructuración modular del código mediante scripts en C#

## Tecnologías utilizadas

- C#
- Unity

## Objetivo del proyecto

Aplicar fundamentos de programación como:
- Tipos de datos
- Operadores aritméticos
- Control de flujo
- Clases y métodos
- Gestión de eventos

## Cómo ejecutar el proyecto

1. Abrir el proyecto desde Unity Hub
2. Cargar la escena principal
3. Ejecutar en modo Play

## Conceptos aplicados

- Programación orientada a objetos (POO)
- Separación de lógica y presentación
- Manejo de estados
- Gestión de errores básicos (división por cero)
